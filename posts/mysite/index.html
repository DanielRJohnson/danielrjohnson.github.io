Welcome to my site

I have finally decided to make my very own website where I can show off my projects and talk my mind about everything I do related to coding. I will probably be gushing about Glimmer, my most recent project where I have been writing an interpreter for my own programming language, but I also plan on writing about school and the space as a whole.

Tech

As someone with experience in industry web development, I had the urge of “if you don’t build this from scratch, you’re fake”, but I decided to shrug this off since another full length project is not what I want on my plate right now. Save that for the office.

So, I spent my Friday evening looking through static site generators, as any sane person would do. I eventually came to the one that you see here, Hugo. Upon finding the right theme, it has made this process very easy! The theme I ended up choosing is LoveIt, which can be found here.

It allows me to be writing this in markdown with cool features like tags, categories, automatic dating, etc. Many times I find tightly curated stacks like this to be too restrictive and not transferable, but Hugo has really impressed me.

Along with this, I am currently working on hosting this on GitHub Pages. One day I might write a full-blown thesis on my disenchantment with monetized hosting platforms, but for now it is suffice to say that I love the convenience of just pushing my code and having it automatically reflect on the hosted site.

Proverbial Gunshot

That being said, this is my site’s very own “Hello, World!”
